# Custom Page Content — Taj Alameera

Ready-to-paste HTML for the 11 empty Salla custom pages. Styling comes automatically
from `salla-inject.js` (the `.content-entry` brand CSS). Just paste the HTML into
each page's rich-text editor (HTML/source mode) on Salla.

## How to paste
1. Salla dashboard → الصفحات (Pages) → open the page.
2. In the editor, switch to **HTML / source view** (`< >` icon).
3. Paste the file contents (skip the top `<!-- comment -->` line — optional).
4. Save. Open the page on the storefront to verify styling.
