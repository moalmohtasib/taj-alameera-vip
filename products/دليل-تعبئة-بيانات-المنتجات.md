# دليل تعبئة بيانات المنتجات — تاج الأميرة

مرحباً 👋 هذا الملف يشرح كيف تعبّي بيانات كل قطعة ذهب عشان نرفعها على المتجر بسرعة وبدون أخطاء.

افتح ملف **OWNER-PRODUCT-DATA** في Google Sheets، وعبّي **صف واحد لكل منتج**. كل عمود مشروح تحت.

---

## شرح الأعمدة (بالترتيب)

| العمود | وش أكتب فيه | مثال |
|---|---|---|
| **رقم المنتج** | نفس رقم مجلد الصور (product-1 = 1) عشان نربط الصورة بالبيانات | `21` |
| **اسم المنتج (عربي)** | اسم القطعة بالعربي، واكتب العيار داخل الاسم | `خاتم ذهب عيار 21 أصفر` |
| **اسم المنتج (انجليزي)** | نفس الاسم بالانجليزي (اختياري) | `21k Yellow Gold Ring` |
| **التصنيف** | نوع القطعة | خواتم / أساور / سلاسل / أطفال / سبائك |
| **العيار** | عيار الذهب | `24` أو `21` أو `18` |
| **لون الذهب** | لون الذهب | أصفر / أبيض / وردي |
| **الوزن بالجرام** | وزن القطعة بالجرام (مهم جداً — يحسب السعر) | `5.20` |
| **أجرة الصنعة (نوع)** | هل الأجرة لكل جرام ولا مبلغ ثابت؟ | `لكل جرام` أو `ثابت` |
| **أجرة الصنعة (قيمة)** | قيمة الأجرة بالريال | `35` (يعني ٣٥ ريال لكل جرام) |
| **الأحجار (نوع)** | إذا فيه أحجار اكتب نوعها، وإلا اكتب لا يوجد | `الماس` / `لا يوجد` |
| **الأحجار (عدد/قيراط)** | عدد الأحجار ووزنها بالقيراط | `12 حجر / 0.50 قيراط` |
| **المقاس** | مقاس الخاتم أو طول السلسلة (إن وجد) | `17` أو `45 سم` |
| **الكمية المتوفرة** | كم قطعة عندك من هذا المنتج | `1` |
| **رقم المنتج SKU** | كود داخلي — نتركه لنا نعبّيه، أو اتركه فاضي | `R-21Y-0520` |
| **الوصف (عربي)** | وصف يبيع القطعة، سطرين ثلاثة | `خاتم فاخر عيار 21، تصميم عصري ناعم يناسب الإطلالة اليومية.` |
| **ملاحظات** | أي شي مهم تبي تقوله لنا | `الصورة الرئيسية رقم 2` |

## معلومة مهمة عن السعر 💰

**ما تكتب سعر نهائي.** السعر يتحسّب تلقائي على المتجر بالمعادلة:

```
السعر النهائي = ((سعر جرام الذهب اليوم حسب العيار × الوزن) + أجرة الصنعة) + ضريبة القيمة المضافة ١٥٪
```

سعر الجرام يجي **مباشر ولحظي** من السوق (نفس مصدر شريط الأسعar فوق الموقع). يعني لو تغيّر سعر الذهب، سعر القطعة يتحدّث لحاله. أنت بس عبّي:
- **الوزن** (بالجرام)
- **العيار** (24/21/18)
- **أجرة الصنعة** (لكل جرام أو ثابت)

وإحنا نتكفّل بالباقي.

## بعد ما تخلّص

1. تأكد إن كل منتج له **رقم يطابق رقم مجلد صوره**.
2. احفظ الملف (Google Sheets يحفظ تلقائي).
3. بلّغنا — إحنا نستورد كل المنتجات على سلة دفعة وحدة، ونربط الصور، ونضبط السعر اللحظي.

أي عمود ما فهمته، اتركه فاضي وبلّغنا. 🌟
